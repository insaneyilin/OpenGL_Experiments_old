# OpenGL_Experiments_old
Some OpenGL experiments written in opengl 1.0.

These codes are my solutions to programming assignments of the course *Computer Graphics* in Zhejiang University. 

The contents are as follows:

1. Colored Six-Ray Star  
2. Matrices in OpenGL
3. 3D Viewing
4. Display List
5. Culling and Illumination
6. Texture
7. Vertices Array, Picking
